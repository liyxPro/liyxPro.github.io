软件工程和软件过程

1、软件生命周期 —— 面试题&重点

“三个时期八个阶段”

(1)软件定义期
1)可行性研究阶段  技术、软硬件、资金、时间、人力、政策...
2)需求分析阶段  功能点列表、非功能性需求(性能、安全、可维护性....)
(2)软件开发期
3)概要设计阶段  划分出模块、职能划分、交互接口、数据说明、网络/硬件设计、技术选择 —— 架构师
4)详细设计阶段  每个模块内部细节、文件、对象、属性和方法... —— 项目设计师
5)开发编码阶段  根据需求文档和设计图编写代码 —— UI/前端/后端合作
6)测试阶段  尽可能发现项目中的BUG —— 测试工程师
(3)软件维护期
7)部署阶段  从开发环境移植到生产环境 —— 运维工程师
8)维护阶段  修改BUG，功能调整
自学的扩展内容：http://www.php100.com/html/shipinjiaocheng/old/

IP地址(西单北大街108号)：网络中用于唯一标识一台计算机的一个地址
如： 172.163.100.89、 127.0.0.1(表示本机)

如何查看自己的计算机的IP地址： Windows+r => cmd => ipconfig

域名(恭王府)：网络中对某个计算机取的特定的别名，便于记忆
如：www.taobao. com、 localhost(表示本机)

DNS服务：网络中可以负责把域名转换为IP地址的服务器

访问远程服务器上的网页

(1)服务器端：　

安装一款Web服务器软件，如Apache Httpd、MS IIS、NginX。
下载地址：xampp.org

(2)服务器端：　

编写网页，保存在特定的目录下。
默认必须保存在c:/xampp/htdocs下

(3)服务器端：　

启动Web服务器。

(4)客户端：　

浏览器地址栏中输入Web服务器的域名或IP地址
